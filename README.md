## 📝 소개

**hoBIT**은 정보대학 챗봇입니다.

**(ho**: 호랑이, **BIT**: 정보의 최소 단위)

정보대학 행정 관련 질문, 정보대학 선배들의 조언들을 한 곳에 모아두었습니다.

## ENV 변수

`hobit` 에 사용되는 환경변수:

| 변수명               | 설명                      |
| -------------------- | ------------------------- |
| `TIMEZONE`           | 서버의 기본 시간대        |
| `DB_HOST`            | 데이터베이스 호스트 주소  |
| `DB_USER`            | 데이터베이스 사용자명     |
| `DB_PASSWORD`        | 데이터베이스 비밀번호     |
| `DB_NAME`            | 데이터베이스 이름         |
| `JWT_SECRET`         | JWT 토큰 서명 키          |
| `JWT_EXPIRATION`     | JWT 액세스 토큰 만료 시간 |
| `HOBIT_NLU_ENDPOINT` | RASA 서버 엔드포인트      |
